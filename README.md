## Project Description

This project performs auditing tasks on a Linux system using shell scripts. It analyzes system information, installed packages, disk permissions, logs, and generates a manifest.

## Scripts Overview

* script1_system_identity.sh → Displays system information
* script2_package_inspector.sh → Lists installed packages
* script3_disk_permission_auditor.sh → Checks file permissions
* script4_log_analyzer.sh → Analyzes system logs
* script5_manifest_generator.sh → Generates audit report

## Requirements

* Linux OS (Ubuntu recommended)
* Bash shell
